# 📊 Mutual Fund Analysis Dashboard - Power BI

## 📌 Project Overview
This project is an interactive **Mutual Fund Analysis Dashboard** created using Power BI.  
It helps analyze fund performance, risk factors, and investment insights for better decision-making.

---

## 🎯 Objectives
- Evaluate mutual fund performance
- Compare funds based on risk and returns
- Analyze fund growth over time
- Provide investment insights using data visualization

---

## 📂 Dataset Features
The dataset includes:
- Scheme Name
- Minimum SIP & Lumpsum
- Expense Ratio
- Fund Size (in Crores)
- Fund Age (Years)
- Fund Manager
- Risk & Return Metrics (e.g., Sortino Ratio)

---

## 📊 Dashboard Insights
Key insights covered:
- 📈 Fund Performance Analysis
- ⚖️ Risk vs Return Comparison
- 🏆 Top Performing Funds
- 📉 Expense Ratio Impact
- 📊 Fund Size & Growth Trends

---

## 🛠️ Tools & Technologies Used
- Power BI
- Power Query (Data Transformation)
- DAX (Data Analysis Expressions)

---

## 🚀 Features
- Interactive slicers and filters
- Dynamic comparison visuals
- Clean and professional dashboard design
- Drill-down analysis
